    Introduction to machine learning

    Machine learning can be defined in opposition to traditional software engineering. Traditional software engineering consists of solving a problem, specifying how the computer should do it. Machine learning consists of specifying algorithms which can be thaught how to solve problem by utilizing data. This site is about machine learning algorithms, the types and how to apply them using python. It basically consists of how to solve problems without knowing what the actual steps of the solution should be.
